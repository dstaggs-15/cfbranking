🏈 College Football Top 25 Rankings

This model is 100% data-driven — no polls, no opinions. Each team is evaluated using:

  • Win Percentage – rewards consistent success.
  • Average Scoring Margin – measures dominance in wins.
  • Strength of Schedule (SoS) – adjusts for opponent quality.

These metrics combine into a composite score:
0.55 × Win% + 0.30 × SoS + 0.15 × (Margin ÷ 25)

Data updates automatically each week from the CFBD API.
